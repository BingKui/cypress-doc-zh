## 简而言之

Cypress 是为现代网络而构建的下一代前端测试工具。我们解决开发人员和质量保证工程师在测试现代应用程序时面临的关键难题。

我们使它很简单：

* 设置测试
* 编写测试
* 运行测试
* 调试测试

Cypress 经常和 Selenium 相提并论；然而，Cypress 在结构和基础上与之有所不同。Cypress 不受 Selenium 限制。

这能够使你编写更快、更简单和更可靠的测试。
